# MicroPowerMeter

This repo contains the Hardware design of my version of a low current meter (from nanoamperes to milliamperes).

More Info [Blog Entry](https://en.laboratoriogluon.com/projects/micropowermeter/).

## Capabilities

- 4 kSps
- 3 Ranges: nA, uA and mA.
- Automatic range switching
- USB connection.
- Low Burden Voltage:
    - 10 uV/nA 
    - 10 uV/uA
    - 10 uV/mA
